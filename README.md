# 声笺 v1.2

让文字被听见。声笺是一款支持 Edge TTS、CosyVoice 3 和 Kokoro 的文本转语音工作台。

## 功能特点

- v1.2 新增极简首页：输入文字即可用默认配置直接创建任务
- 首页支持保留草稿进入高级工作台，形成“快速开始 → 精细配置 → 任务管理”业务流
- 全新“纸笺 + 声波”品牌 Logo，统一首页、工作台、登录页和后台视觉
- 新增 JSONL 结构化运行日志与独立审计日志
- 后台日志中心支持类型、级别、关键词筛选和原始日志下载
- 日志等级、单文件容量和滚动备份数量可由管理员配置
- v1.1 新增统一 TTS Provider 层，可在 Edge TTS、CosyVoice 3、Kokoro 间切换
- CosyVoice 3 与 Kokoro 通过本机 sidecar 部署，模型运行环境可独立升级
- 本地 Provider 支持健康检查、动态音色发现、能力声明和任务来源追踪
- 本地服务地址仅允许回环地址，避免把生成接口暴露到外部网络
- 支持多种中文语音角色：
  - 普通话：晓晓、晓伊、云健等
  - 方言：东北话（晓北）、陕西话（晓妮）
  - 粤语：晓佳、晓曼、云龙
  - 台湾腔：晓辰、晓语、云哲
- 支持自定义文件名
- 管理员配置默认目录后可由后端直接保存 MP3 或批量 ZIP
- 支持语速、音量和音高配置
- 美观的用户界面
- 实时显示语音角色信息
- 支持长文本转换（上限200000字符）
- 支持长文本自动分段，按自然断点逐段转换
- 长文本分段完成后自动合并为单个 MP3 下载
- 单条文本转换已任务化，支持进度、预计剩余时间、取消任务和失败片段重试
- 使用受控后台队列，限制并发请求并降低服务限流风险
- 支持试听前 300 字，快速确认音色和语速
- 支持生成后页面内播放音频
- 支持最近生成历史记录、播放、下载和删除
- 收藏、最近使用、历史记录和已完成任务可在重启后恢复
- 首次启动创建本地管理员，支持登录、退出和会话验证
- 管理员可创建、禁用、授权用户及重置密码
- 普通用户的任务、历史和音频互相隔离
- 前台聚焦创作与个人任务中心，系统设置、诊断、用户和全局任务移至后台
- 任务中心支持刷新恢复、多任务 FIFO 排队和总任务/分项多维时间进度
- 支持收藏完整的音色、语速、音量和音高配置组合
- 批量 TXT 可分多次追加选择，并自动忽略重复文件
- 支持用户级自动下载策略
- 支持音色搜索、收藏、最近使用排序
- 支持管理员在线刷新中文音色目录，离线时使用本地缓存
- 支持短视频口播、新闻播报、小说旁白、儿童故事、粤语口播等场景预设
- 支持设置页：默认音色、默认语速、代理、分段长度、临时文件保留、自动打开浏览器
- 支持网络和代理诊断
- 支持导出脱敏诊断包和本地滚动日志
- 网络诊断带短时缓存，避免弱网下重复阻塞操作
- 支持动态端口和单实例启动，端口占用时会自动选择可用端口
- 支持批量 TXT 转语音
  - 一次最多上传20个 TXT 文件
  - 每个 TXT 文件最多20000字符
  - 支持逐个下载 MP3 或打包下载 ZIP
- 自动重试机制
- 使用受控文件 token 下载音频，避免暴露本机临时路径
- 应用专用临时目录和过期文件清理
- 统一 API 错误响应，便于前端展示和后续测试

## 快速开始

### 使用打包版本（推荐）

1. 从 [Releases](https://github.com/keliihall/edge-tts/releases) 页面下载对应系统的安装包：
   - Windows: `ShengJian-v1.2-Windows.zip`
   - macOS: `ShengJian-v1.2-macOS.zip`
   - Linux: `ShengJian-v1.2-Linux.tar.gz`

2. 解压下载的文件：
   - Windows: 解压后双击 `shengjian.exe`
   - macOS: 解压后在终端运行 `./shengjian`
   - Linux: 解压后运行 `./shengjian`

应用会自动打开默认浏览器访问界面。

macOS 首次运行若被系统阻止，可在“系统设置 → 隐私与安全性”中允许打开。当前发布包为未签名的单文件程序。

### 从源码运行

1. 克隆仓库：
```bash
git clone https://github.com/keliihall/edge-tts.git
cd edge-tts
```

2. 创建虚拟环境：
```bash
python3 -m venv .venv
source .venv/bin/activate  # Linux/Mac
# 或
.\.venv\Scripts\activate  # Windows
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

4. 运行应用：
```bash
python run.py
```

应用会自动打开默认浏览器访问界面。如果没有自动打开，请访问 http://127.0.0.1:5013

首次打开会进入管理员创建页。账户仅保存在本机 SQLite 数据库中，密码使用安全哈希保存。

登录后 `/` 为极简快速开始首页，`/studio` 为完整创作工作台。首页仅保留文字输入、
“立即生成”和“高级设置”：前者使用管理员设置的默认引擎和音色，后者会把当前草稿带入工作台。

### 启用本地 TTS

1. 按 [本地 TTS 部署协议](docs/LOCAL_TTS.md) 启动 CosyVoice 3 或 Kokoro sidecar。
2. 管理员进入“系统设置”，启用对应引擎并填写本机服务地址。
3. 在诊断页确认引擎状态为“可用”，保存默认引擎和音色。
4. 回到工作台，在“语音引擎”中选择本地模型。

默认地址为 CosyVoice `http://127.0.0.1:50000`、Kokoro `http://127.0.0.1:50001`。
模型权重和 GPU 运行时不打包进声笺安装包。

### 开发与测试

```bash
pip install -r requirements-dev.txt
pytest -q
python scripts/release_check.py
```

联网环境可额外执行真实服务冒烟验证：`python scripts/smoke_tts.py`。

## 使用说明

### 单条文本转换

1. 在文本框中输入要转换的文本
2. 从下拉菜单中选择语音角色、语速、音量和音高
3. 可选：使用搜索、收藏或场景预设快速选择音色
4. 可选：点击"试听"试听前 300 字
5. 点击"转换为语音"按钮创建转换任务
6. 查看分段进度、成功/失败数量和预计剩余时间
7. 可选：取消任务，或在失败后重试失败片段
8. 转换完成后可在页面内播放音频
9. 可选：修改下载文件名
10. 点击"下载音频"按钮保存文件；多段结果会自动合并为单个 MP3

### 历史记录

1. 转换成功后会自动进入"最近生成"
2. 可直接播放首个成功音频片段
3. 可重新下载音频或删除历史记录

### 设置与诊断

1. 管理员在"设置与诊断"区域配置默认音色、默认语速和代理地址
2. 可调整历史保留天数、临时文件保留小时、长文本分段字数
3. 可选择启动后是否自动打开浏览器
4. 点击"重新诊断"查看代理、Edge TTS 服务、设置文件和临时目录状态
5. 点击"导出诊断包"生成不包含文本、密码和音频的排查 ZIP

### 用户管理

1. 管理员从顶部导航进入"管理"
2. 创建普通用户或其他管理员
3. 可启用、禁用、调整角色或重置密码
4. 系统始终保留至少一个启用的管理员

### 批量 TXT 转换

1. 切换到"批量 TXT"模式
2. 选择语音角色和语速
3. 选择一个或多个 `.txt` 文件
4. 点击"批量转换"按钮
5. 等待队列逐个完成
6. 点击单个"下载"保存对应 MP3，或点击"下载 ZIP"打包保存全部成功文件

```

├── app.py              # Flask API、任务队列和 TTS 流程
├── auth.py             # 本地认证与管理员接口

│   ├── css/style.css   # 响应式界面样式
│   ├── fonts/          # Noto Sans SC 字体与开源许可证
```

### 打包说明

项目使用 PyInstaller 进行打包，支持生成各平台的独立可执行文件：

构建前请先安装开发依赖：`pip install -r requirements-dev.txt`。

- Windows:
```bash
.\build_windows.bat
```

- macOS:
```bash
./build_macos.sh
```

- Linux:
```bash
./build_linux.sh
```

打包后的文件位于 `dist` 目录。

## 技术栈

- 后端：Flask
- 语音转换：Edge TTS + CosyVoice 3/Kokoro 本地 sidecar
- 前端：HTML5, CSS3, JavaScript
- 字体：Noto Sans SC（本地打包，SIL Open Font License）

## 注意事项

- 单条文本长度限制为200000字符，系统会自动分段转换
- 批量转换限制为单次20个 TXT 文件，每个文件20000字符以内
- 语速支持0.75x、0.9x、1.0x、1.1x、1.25x、1.5x、2.0x，默认1.0x
- 音频文件格式为 MP3
- 转换后的临时文件会按设置自动清理
- 日志仅记录请求元数据、任务 ID、参数标识和操作结果，不记录输入正文、密码或音频
- 历史任务在对应临时音频过期后仍保留元数据，但需要重新生成才能再次下载
- 如果 5013 端口被占用，应用会自动选择后续可用端口
- 如果已有实例正在运行，重复启动会直接打开已有实例
- 支持的语音角色由当前 Provider 动态提供
- Edge TTS 需要联网；CosyVoice 3 与 Kokoro 可完全离线运行
