# 声笺 v1.0 发布验收

本文档定义 v1.0 的交付边界。所有必需项均已实现，并由自动化测试、真实服务冒烟或浏览器验证覆盖。

## 核心能力

- [x] 单条文本异步转换、自然分段、进度、预计剩余时间、取消和失败片段重试
- [x] 分段 MP3 自动合并，支持页面播放、下载和后端默认目录保存
- [x] 批量 TXT 队列、取消、单项重试、单项下载、ZIP 下载和后端保存
- [x] 语速、音量、音高、场景预设、试听、搜索、收藏和最近使用
- [x] 管理员在线刷新中文音色，离线使用内置目录与本地缓存
- [x] 最近生成、历史保留、重启恢复和过期音频清理
- [x] 前台任务中心在刷新后恢复任务，并展示总任务与分项时间进度
- [x] 多任务按创建时间进入 FIFO 队列，后台可查看、取消和清理全局任务
- [x] 配置收藏夹保存完整生成参数，批量文件支持多次追加与去重
- [x] 用户级自动下载策略避免轮询和刷新造成重复下载

## 管理与安全

- [x] 首次运行创建管理员，登录、退出、会话验证和登录失败限流
- [x] 用户创建、启用/禁用、角色调整、密码重置和最后管理员保护
- [x] 普通用户任务、历史、试听、播放和下载隔离
- [x] 管理设置、动态音色刷新和默认目录保存仅管理员可用
- [x] SQLite 持久化、受控文件路径、音色白名单和统一 API 错误
- [x] 诊断包不包含输入文本、密码哈希或音频

## 工程与发布

- [x] 动态端口、单实例启动、滚动日志、请求 ID 和网络诊断缓存
- [x] 后端拆分认证与存储模块，前端脚本从模板中独立
- [x] API 契约、pytest 回归测试、三平台 CI 和统一 release gate
- [x] PyInstaller 单文件构建脚本及带版本号的发布压缩包
- [x] 桌面与 390px 移动视口交互和无横向溢出验证
- [x] 前台创作/任务与后台用户/任务/设置/诊断完成信息架构拆分
- [x] 前台收拢为单一创作工作台，配置、参数、下载策略与任务进度保持统一视觉层级
- [x] 前台重构为“声音与输出 + 内容创作”两步工作台，提交区实时回显当前生成配置
- [x] 移动端默认折叠完整配置并保留配置摘要，缩短默认参数下的高频创作路径
- [x] 前台、后台与认证页统一应用科技蓝主色、冷灰蓝底色和语义状态色
- [x] 产品统一更名为“声笺”，发布产物统一使用 ShengJian/shengjian 命名
- [x] Noto Sans SC 本地打包并附带 SIL Open Font License
- [x] 真实 Edge TTS 生成、合并与 ffprobe 音频解析验证

## v1.0 非目标

以下能力不影响本地单机版核心闭环，留作后续版本：

- 系统托盘、原生安装器、代码签名与自动更新
- 云端账户、远程访问、多人协作与付费能力
- 音频波形编辑、背景音乐混音和专业后期处理
