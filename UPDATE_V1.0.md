# 声笺 v1.0 更新说明

“声笺”取文字成笺、落字成声之意。v1.0 是首个稳定版本，重点是把已有能力收口为适合日常使用和分发的本地文本转语音工具。

## 主要变化

- 长文本仍按自然断点分段生成，但下载时自动合并为单个 MP3。
- 使用固定大小的后台任务池，避免同时创建过多线程和外部请求。
- 收藏、最近使用、历史记录和任务状态均会持久化，重启后可恢复。
- 状态存储升级为 SQLite，文本与批量任务均可在重启后恢复。
- 批量任务支持取消、失败文件单独重试、尝试次数和耗时展示。
- 首次启动创建本地管理员，支持登录、退出和登录失败限流。
- 管理员可创建、禁用、授权用户及重置密码，并保护最后一个管理员。
- 普通用户的任务、历史、试听和下载权限互相隔离。
- 增加音量、音高微调，场景预设可同时覆盖语速、音量和音高。
- 管理员可在线刷新中文音色目录，结果写入本地缓存供离线启动。
- 状态和设置采用原子写入，降低异常退出导致配置损坏的风险。
- 网络诊断增加短时缓存，并将连接探测超时收紧到 3 秒。
- 批量和单条下载统一校验受控临时目录，收紧文件访问边界。
- 页面改为更聚焦的工作台布局，增加页内导航、键盘焦点和非阻塞提示。
- 页面版本号、文本上限和批量文件上限由后端统一注入，避免前后端配置漂移。
- 诊断信息改用安全 DOM API 渲染，不再拼接可注入 HTML。
- 增加滚动结构化日志、请求 ID 和脱敏诊断包导出。
- 增加 API 契约、三平台 CI 和统一 release gate。
- 前台重构为创作、偏好和个人任务中心，后台独立承载全局任务、用户、设置和诊断。
- 任务与分项展示创建、开始、耗时、预计剩余、结束时间及排队位置，刷新后自动恢复。
- 队列改为严格 FIFO，允许连续创建多个任务并按创建顺序执行。
- 配置收藏夹可保存整套生成参数，批量 TXT 支持多次追加选择和去重。
- 增加用户级自动下载策略，并避免刷新或轮询导致重复下载。
- 前台视觉重构为开阔的单一创作工作台，常用配置、生成参数与下载策略集中管理。
- 任务中心改为轻量进度列表，并针对桌面和 390px 移动视口统一信息密度与操作布局。
- 前台信息架构升级为声音配置与内容创作两步工作流，桌面双栏并行、移动端配置摘要折叠。
- 提交区实时显示当前音色、语速、音量和音高，减少创建任务前的配置确认成本。
- 前台、后台和登录设置页统一为科技蓝视觉系统，保留成功、警告和错误的语义区分。
- 产品正式命名为“声笺”，可执行文件与发布包采用 ShengJian/shengjian 命名。
- 界面字体统一为本地打包的 Noto Sans SC，改善中文跨平台显示一致性。

## 验证

- 39 项后端自动化测试覆盖转换、长文本、批量、任务恢复、队列顺序、配置收藏、下载安全、设置、认证、权限隔离、诊断、SQLite 持久化和失败重试。
- 前端状态测试覆盖批量文件的追加、去重、保序和上限处理。
- 使用 PyInstaller 6.16.0 和 Python 3.13.3 完成 macOS arm64 单文件构建，并验证首次设置页、静态资源、健康接口和单实例行为。
- 对桌面和 390px 移动视口执行页面渲染与交互检查，页面无横向溢出。
- 验证文本/批量切换、收藏配置弹窗和任务分项展开，浏览器控制台无错误或警告。
- 使用真实文本任务验证创建、处理中、完成、下载与任务卡布局，并在验收后清理测试数据。
- 使用真实 Edge TTS 服务验证在线音色刷新、音量/音高转换和后端目录保存，生成结果通过 `ffprobe` 解析。

## 兼容性

- Python 3.9 及以上。
- Windows、macOS 和 Linux 可从源码运行。
- 打包产物需在对应操作系统上构建。
